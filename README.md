# SPOTLIGHT FILMS（合同会社サイト）

合同会社 **SPOTLIGHT FILMS** の公式サイトを、このフォルダで管理します。

- **公開URL**: `https://spotlight-films.jp/`
- **映画『残すということ』LP（別フォルダ）**: `~/Desktop/nokosu-film-lp`

## ローカル起動

```bash
npm install
npm run dev
```

ブラウザで `http://localhost:3000` を開きます。
